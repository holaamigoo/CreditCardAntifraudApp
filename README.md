# CreditCardAntifraudApp
# 🛡️ Система обнаружения мошеннических транзакций

## 📋 Описание проекта

Веб-приложение для обнаружения мошеннических транзакций, помогает выявлять подозрительные операции. Проект объединяет методы машинного обучения с интуитивно понятным веб-интерфейсом, позволяя пользователям загружать данные, получать предсказания и вероятностную оценку мошенничества каждой транзакции и анализировать результаты.

### Основные возможности

- **Используется модель машинного обучения**: XGBoost Classifier
- **Веб-интерфейс** на Streamlit с тремя режимами работы:
  - Ручной ввод транзакций
  - Загрузка CSV-файлов
  - История предсказаний
- **Интерактивная визуализация** результатов (распределение вероятностей, доля мошеннических транзакций, важность признаков)
- **Цветовая кодировка** риска мошенничества (зелёный/оранжевый/красный)
- **Сохранение истории** в SQLite с возможностью экспорта в CSV
- **Настраиваемый порог** классификации и цветовой кодировки
